## Rules

- Every integration must have an owner.
- Every integration must have a contract.
- Every integration must define permissions.
- Deprecated integrations require migration guidance.
- Integrations must not bypass governance.

## Validation Checklist

- [ ] Registry entries have IDs.
- [ ] Owners are defined.
- [ ] Contracts are linked.
- [ ] Status is current.
- [ ] Deprecated items have migration path.
